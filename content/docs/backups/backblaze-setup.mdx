# Setup

A guide to setup the backblaze and get your keys

## Create a Bucket

![create-bucket](../images/backblaze-setup/create_bucket_1.png)

![create-bucket](../images/backblaze-setup/create_bucket.png)

![bucket-created](../images/backblaze-setup/endpoint.png)

Here you can see the:

This is s3-compatible endpoint

**EndPoint**: s3.eu-central.backblazeb2.com

**Region**: eu-central

## Create Application keys

Go To

![Application-key-button](../images/backblaze-setup/application_key.png)

Choose Application keys and then *Add a New Application Key*

![add-application-key](../images/backblaze-setup/create_application_key.png)

![create-Application-key](../images/backblaze-setup/new_key.png)

Choose the appropriate bucket you want to give access to.
Make sure:

*Type of Access* : Read and write both

Avoid configuring other options

![application-keys](../images/backblaze-setup/your_key.png)

!!Hurray we got the keys

We already got:

**EndPoint**: **s3.eu-central.backblazeb2.com**

**Region**: **eu-central**

**Bucket Name** (that we set): **back-bucket**

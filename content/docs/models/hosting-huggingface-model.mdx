# Hosting a Huggingface Model

Step-by-step guide to hosting HuggingFace models on Aquanode.

You can host text-to-text HuggingFace models on Aquanode seamlessly via vllm engine.

1. Go to **Model Pipelines** section under **WORKLOADS**

2. Enter the configuration like:
- model repo url
- HF TOKEN
- API KEY
and other settings you want to config.

3. Select an appropriate GPU according to your model and config

4. **Deploy**

You will get a link to request to, with you API_KEY

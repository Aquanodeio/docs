title: Supported Storage Providers
description: Aquanode provides built-in storage for snapshots and persistent data across deployments.
---

Aquanode uses its own built-in storage for snapshots and data persistence — no external storage configuration needed.

## Setup

1. Go to [Settings](https://console.aquanode.io/settings) → **Integrations**
2. Storage is pre-configured by Aquanode
3. Use in deployment configuration for:
   - Automated snapshots
   - Data persistence across deployments

Deploy your Node.js and Python applications directly from Github repositories on Aquanode's infrastructure with automated builds and deployments.

## Deployment Process

### Step 1: Access Github Deployment

Navigate to the Services section in your Aquanode dashboard.[Github Deployment](https://console.aquanode.io/workloads/services/github)

### Step 2: Select Github Repo Tab

Choose the **Github Repo** tab to begin configuring your repository deployment.

### Step 3: Configure Your Repository

#### A. Repository Configuration

- #### Public repository 
  - **Github Repository Link**: Enter public your repository URL

- #### Private Repository
  1. **Connect Github Account**: Click on the "Connect Github" button to authorize Aquanode to access your private repositories.
  2. **Select Repository**: After connecting, select the desired private repository from the dropdown list.
  3. **Branch Selection**: Choose the branch you want to deploy (e.g., `main`, `development`).

#### B. Application Configuration

- **Install Command**: Commands to install dependencies (e.g., `npm install`, `pip install -r requirements.txt`)
- **Run Command**: Commands to start your application (e.g., `npm start`, `python app.py`)
- **Environment Variables**: Configure runtime settings and secrets

#### C. Machine Configuration

- **CPU & Memory**: Allocate resources based on your application needs
- **Storage**: Persistent storage for application data
- **Port Configuration**: Configure network access for your application

### Step 4: Deploy on GitHub

Review your configuration and click **Deploy on GitHub**.

Hurray! Your application is now deployed and running.

## Supported Applications

Currently we support **Node.js** and **Python** applications for Github repository deployments.

# Deploying through Github

Learn how to deploy applications directly from your Github repositories.

We currently support node and python applications hosting via GitHub Repositories

1. Go to service section

2. Choose **Github Repo** tab

3. Enter you github repo link and other configs

4. Install command, Run command, github token for private repo, etc.

6. click on **Deploy on GitHub**

Hurray!! We did it.

*Note: Currently we support only **node** and **python** applications.*

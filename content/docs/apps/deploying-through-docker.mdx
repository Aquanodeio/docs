# Deploying through Docker

Deploy your applications using Docker containers on Aquanode.

1. Go to Services inside Workloads section.

2. Select Docker Image Tab

*Page Like this will open, Enter your desired config here*

![docker-config](../images/apps/docker_config.png)

Chose the GPU you need from our wide variety.

**Deploy Custom Instance**

*You are good to go*

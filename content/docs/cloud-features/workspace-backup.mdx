## Prerequisites

Backups in Aquanode currently require **S3-compatible object storage**.  
Before enabling backups, configure your storage credentials.

### Supported Providers

- **AWS S3**  
  - [Get Started with S3](https://docs.aws.amazon.com/AmazonS3/latest/userguide/GetStartedWithS3.html)  
  - [Create a Bucket](https://docs.aws.amazon.com/AmazonS3/latest/userguide/create-bucket-overview.html)  

- **BackBlaze B2**  
  - [Get Started with BackBlaze B2](https://www.backblaze.com/docs/cloud-storage-get-started-with-the-ui)  
  - [Enable BackBlaze B2](https://www.backblaze.com/docs/cloud-storage-enable-backblaze-b2)  

- **MinIO**  
  - Self-hosted, deployable on Kubernetes or bare metal  
  - [Installation Guide](https://docs.min.io/community/minio-object-store/operations/deployments/installation.html)
